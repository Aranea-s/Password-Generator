# Password-Generator
A simple password generator written on C++

## Features
- Generates strong random passwords
- Customizable password length (1-63 if spec_char is false, 1-90 if spec_char is true)
- Includes letters, numbers, and symbols

## example of output
```
==================================
Service: ...
login: ...
Password: 3nMSGrE7HqaXg4TYFJtU5fjZ
```
